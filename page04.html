Model Deployment

Below is a Python script used in this project along with an explanation:

Script 1 : Model_Deploy.py

	import numpy as np
	import pickle
	import streamlit as st
	
	# Loading the saved model
	loaded_model = pickle.load(open(r"C:\Users\farzan\Desktop\diabetes\trained_model.sav", 'rb'))
	
	def diabetes_func(input_data):
	
		# Changing the input data into numpy array
		input_data_np = np.asarray(input_data)
	
		# reshaping the array
		input_data_reshaped = input_data_np.reshape(1,-1)
	
		prediction = loaded_model.predict(input_data_reshaped)
		print(prediction)
	
		if (prediction[0] == 0):
			return 'No Diabetes Detected'
		else:
			return 'Diabetes Detected'
			
	def main():
	
		# Title
		st.title('Diabetes Prediction')
	
		Pregnancies = st.text_input("Number of Pregnancies")
		Glucose = st.text_input("Glucose Level")
		BloodPressure = st.text_input("Blood Pressure Value")
		SkinThickness = st.text_input("Skin Thickness")
		Insulin = st.text_input("Insulin Level")
		BMI = st.text_input("BMI")
		DiabetesPedigreeFunction = st.text_input("Diabetes Pedigree Function Value")
		Age = st.text_input("Age")
	
		diagnosis = ''
	
		if st.button("Test Results"):
			diagnosis = diabetes_func([Pregnancies,Glucose,BloodPressure,SkinThickness,Insulin,BMI,DiabetesPedigreeFunction,Age])
	
		st.success(diagnosis)
	
	
	if __name__ == '__main__':
		main()

Step 1: Imports :
We import necessary libraries, including numpy for numerical operations, pickle for loading the saved model, and streamlit for building the web application.

Step 2: Loading the Saved Model :
We load the pre-trained model using pickle.load. Ensure that the path to the 'trained_model.sav' file is correct on your system.

Step 3: "diabetes_func" Function :
- This function takes an input list of data representing features related to diabetes prediction.
- It converts the input data into a numpy array, reshapes it, and uses the loaded model to make predictions.
- Based on the prediction result, it returns 'No Diabetes Detected' or 'Diabetes Detected'.

Step 4: "main" Function :
- In the main function, we create a Streamlit web application.
- We provide input fields for the user to enter information related to diabetes prediction, such as the number of pregnancies, glucose level, blood pressure, and so on.
- When the user clicks the "Test Results" button, the diabetes_func() function is called with the provided input data, and the diagnosis is displayed.

Step 5: Running the Streamlit App :
We use the if __name__ == '__main__': block to run the Streamlit app when the script is executed directly.

Step 6: Displaying the Diagnosis :
The diagnosis result ('No Diabetes Detected' or 'Diabetes Detected') is displayed using st.success.
